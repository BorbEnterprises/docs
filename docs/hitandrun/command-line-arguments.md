This page documents the various command line arguments that the game has.

# ARTSTATS

# AUDIOSPEW

# CDFILES

# COINS
Makes the player start with 1000 coins when starting a New Game.

# DEBUGBV

# DEMOTEST

# DESIGNER

# DETECTLEAKS

# FEGAGS
Makes the game output to the console which gag it is about to play on the main menu. Also makes the gags play every 15 seconds instead of every 15-30 seconds.

# FEUNJOINED
Makes the game *attempt* to load PNG files instead of using the PNG files inside the frontend P3Ds.

# FILENOTFOUND

# FIREWIRE

# FPS

# HEAPSTATS

# HOSTFILES

# LANGUAGE

# LARGEHEAPS

# MANUALRESETDAMAGE
Repairs the player's car when they reset it onto the road with the reset key.

# MEMCARDCHEAT

# MEMMONITOR

# MUTE
Disables most audio and also seems to prevent settings from being loaded.

# NOAVRIL

# NODIALOG

# NOEFFECTS

# NOFRUITLESS

# NOHAPTIC

# NOHEAPS

# NOHUD
Disables the in-game HUD.

# NOLOADINGSPEW

# NOMUSIC

# NOPEDS
Disables sidewalk pedestrians and makes other types of NPCs invisible.

# NOSPLASH

# NOTRAFFIC
Disables traffic cars.

# NOTUTORIAL

# PCTEST

# PRINTFRAMERATE

# PRINTLOADTIME

# PRINTMEMORY

# PROGSCAN

# PROPSTATS

# RADTUNER

# RANDOMBUTTONS
Simulates random button inputs continuously.

# RELEASEPRINT

# SEQUENTIALDEMO

# SHORTDEMO

# SHOWDYNALOAD

# SKIPFE
Seems to attempt to make the game skip the main menu and go straight in-game but causes it to crash.

# SKIPMEMCHECK

# SKIPMOVIE

# SKIPSUNDAY

# SNPROFILER

# SPEEDOMETER
Shows a text speedometer (the same one as the cheat code).

# TOOL

# WINDOW
