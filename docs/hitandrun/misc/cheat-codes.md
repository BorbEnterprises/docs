This page contains information about the games various cheats and how to enter them.

# Entering Cheats
## On PC
Hold F1 while in the the top level of the Settings menu and enter the code using the arrow keys.

This will crash the game when you enter a valid cheat code on the Main Menu on this platform unless you have "Main Menu Cheat Code Enter" enabled in the [Bug Fixes](../../lucasmodlauncher/hacks/bug-fixes.md#settings_crashes_main-menu-cheat-code-enter-crash) hack included with [Lucas' Simpsons Hit & Run Mod Launcher](../../lucasmodlauncher/intro.md).

## On GameCube / Xbox
Hold L+R in the top level of the Settings menu and enter the code using the face buttons.

## On PlayStation 2
Hold L1+R1 in the top level of the Settings menu and enter the code using the face buttons.

This cheat unlocks all vehicles in the phonebooth and also enables a special menu in the phonebooth containing more cars. This menu can be accessed with F1 on PC, L/R on GameCube and L1/R1 on PlayStation 2.
