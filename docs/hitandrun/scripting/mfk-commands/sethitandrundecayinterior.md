This command was presumably supposed to set the Hit & Run decay rate while in an interior.

However, in reality its actually an alias for [KillAllChaseAI](killallchaseai.md) which is presumably a mistake in the game.

# Notes
For mods that would like to use this command as intended, the [Mod Launcher's](/lucasmodlauncher/intro.md) [Additional Script Functionality](/lucasmodlauncher/hacks/asf/intro.md) hack replaces it with a functional command.

See the [Modified Commands](/lucasmodlauncher/hacks/asf/modified-commands.md#sethitandrundecayinterior) for more information.
