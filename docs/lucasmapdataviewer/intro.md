Lucas' Simpsons Hit & Run Map Data Viewer makes it easy to view map data from The Simpsons Hit & Run. It especcially helps with finding specific instances of chunk types that don't store names such as Fences and Intersects.

# Download
This tool is available to download from the [Donut Team downloads page](https://donutteam.com/downloads/MapDataViewer).
