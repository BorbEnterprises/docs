# This Element
This type of element allows you to define a basic RGB material to be referenced by other types of elements.

```xml
<Material Name="Null" Red="255" Green="255" Blue="255"/>
```

* **Name**: The name of the material.
* **Red**: The red value of the material.
* **Green**: The green value of the material.
* **Blue**: The blue value of the material.

# Child Elements
This type of element has no children.
