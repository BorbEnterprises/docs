# Syntax
```text
<AnimName> [Anim] [<PoseTransformAnimationName>]
```

* **Name**: The name of the Anim.
* **PoseTransformAnimationName**: The name of the [PoseTransformAnimation XML Element](../../../xml-format/elements/animations/posetransformanimation.md) to use for this Anim.
