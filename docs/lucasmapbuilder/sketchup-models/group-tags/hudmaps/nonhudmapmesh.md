This tag makes it so a group's geometry will not be included in the HUD Map [Mesh](../../../xml-format/elements/entities/mesh.md) if it otherwise would be.

# Usage
{{ snippet lucasmapbuilder/sketchup-models/group-tags/usage-include }}

* `HUDMap.xml`

{{ snippet lucasmapbuilder/sketchup-models/group-tags/usage-execute }}

* `BaseHUDMap`

# Syntax
```
GroupName [NonHUDMapMesh]
```

# Group Structure
This tag can be applied to any group.
