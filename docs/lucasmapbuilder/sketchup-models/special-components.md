This page explains the various SketchUp components included with the tool and what they represent.

# BillboardQuad
Used to represent [Old Billboard Quads](../../pure3d/chunk-types/chunk-17002.md).

# Camera
Used with the [AnimatedCamera group tag](group-tags/cameras/animatedcamera.md) to represent a position for the camera.

# Locator
Used to represent [Locators](../../pure3d/chunk-types/chunk-3000005.md) and positions inside groups using [Locator group tags](group-tags/all-tags.md#locator-tags).

# RectTriggerVolume
Used to represent rectangular [Trigger Volumes](../../pure3d/chunk-types/chunk-3000005.md#child-trigger-volume-0x3000006) inside groups using [Locator group tags](group-tags/all-tags.md#locator-tags).

# SphereTriggerVolume
Used to represent spherical [Trigger Volumes](../../pure3d/chunk-types/chunk-3000005.md#child-trigger-volume-0x3000006) inside groups using [Locator group tags](group-tags/all-tags.md#locator-tags).
