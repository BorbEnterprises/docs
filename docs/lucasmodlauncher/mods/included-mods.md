This page explains what the various mods included with the Mod Launcher do.

# No Audio

**Added in Version 1.23.2.**

Disables audio using the game's [MUTE](../../hitandrun/command-line-arguments.md#mute) command line argument.

# No HUD

**Added in Version 1.10.**
	
Disables the HUD using the game's [NOHUD](../../hitandrun/command-line-arguments.md#nohud) command line argument.

# No Introduction Movies

**Added in Version 1.6 and removed in Version 1.15.**

Disabled introduction movies by using the [OccludedPath](../hacks/cf/intro.md#configuration) feature of [Custom Files](../hacks/cf/intro.md).

This was replaced by the [No Introduction Movies](../hacks/no-introduction-movies.md) hack.

# No Traffic

**Added in Version 1.6.**

Disables traffic cars using the game's [NOTRAFFIC](../../hitandrun/command-line-arguments.md#notraffic) command line argument.

# No License Screen Delay

**Added in Version 1.22.**

Removes the delay on the license screen.

# Never Busted

**Added in Version 1.22.**

Makes it so you cannot get busted by the police by using [Custom Trigger Actions](../hacks/custom-trigger-actions.md) to disable the Event that busts you.

# Repair Car On Reset

**Added in Version 1.6.**

Makes your car get repaired when resetting to the road using the game's [MANUALRESETDAMAGE](../../hitandrun/command-line-arguments.md#manualresetdamage) command line argument.

# Speedometer

**Added in Version 1.17.**

Enables the basic speedometer on the HUD using the game's [SPEEDOMETER](../../hitandrun/command-line-arguments.md#speedometer) command line argument.

# Text Names

**Added in Version 1.6.**

When this mod is enabled, the [Custom Text](../hacks/custom-text.md) hack will make the game display internal names of text strings instead of the actual strings.

# Unlock All Rewards

**Added in Version 1.14.**

This mod unlocks all rewards by simply requiring the [Unlock All Outfits](../hacks/unlock-all-outfits.md), [Unlock All Vehicles](../hacks/unlock-all-vehicles.md) and [Unlock Bonus Game](../hacks/unlock-bonus-game.md) hacks.
