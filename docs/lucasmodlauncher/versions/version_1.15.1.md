**This update was released on January 24th, 2017.**

# Launcher
## General
Made the error message shown when failing loading mods ask to show the folder in explorer when possible.

# Mod Features

* Added the `OptionalFramework` property to the `[Miscellaneous]` section.
* Added the `OptionalHack` property to the `[Miscellaneous]` section.
* Added the `RewriteMetaINI` property to the `[Compile]` section.

# Hacks
## Custom Bonus Mission Support
Fixed a crash when launching the game with a non-cracked executable.

## Aspect Ratio Support
Fixed an issue where the background of the credits didn't fully cover the screen in some wider aspect ratios.
