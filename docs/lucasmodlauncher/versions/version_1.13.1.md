# Hacks

## Borderless
Added this new hack. This makes the game window not have a border.

This hack is a good workaround for a bug that occurs on Windows 10 when the game is in fullscreen.

## Cheat Keys
Added a "Continuous" setting to make holding keys act as it did before Version 1.13.
