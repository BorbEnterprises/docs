This page documents community created language packs for the Mod Launcher and how to create new ones.

# Downloading Languages
This is a list of currently available languages and where to obtain them.

Additional languages can be installed simply by placing the XML file into a "Languages" folder next to the Mod Launcher's executable.

| Language               | Author                                                           | Version  | Last Updated        | Download                                                      |
|------------------------|------------------------------------------------------------------|----------|---------------------|---------------------------------------------------------------|
| English                | Donut Team                                                       | Built-in | Built-in            | Built-in                                                      |
| Russian / русский      | [Gordon CMB](https://donutteam.com/@CMB_Union)                   | 1.25     | July 23rd, 2020     | [Download / скачать](https://donutteam.com/forum/topic/2075/) |
| Portuguese / Português | [Charles Montgomery](https://donutteam.com/@mrburns)             | 1.22.2   | April 15th, 2019    | [Download / Baixar](https://donutteam.com/forum/topic/2661)   |

Once installed, the Mod Launcher will prompt you to choose a Language the next time you start it. The setting can be changed again later by going "Open > Launcher Settings..." and going to the "Launcher" tab.

**NOTE**: Donut Team does not validate or take responsibility for any errors or otherwise questionable content in third-party language packs. If you find any such things, please let us know via our support email support@donutteam.com, [our forum](https://donutteam.com/forum/) or our [Discord server](https://donutteam.com/discord).

# Creating Languages
You can start creating your own language localization for the Mod Launcher by downloading the newest template XML file from the table below (currently Template_1.23.4.xml), placing it in a "Languages" folder next to the Mod Launcher's executable and un-commenting and editing the Translation elements inside it.

Any language strings not included in your XML file will fall back to their English defaults. Certain strings also have other fallbacks for backwards compatibility with older versions of the Mod Launcher.
