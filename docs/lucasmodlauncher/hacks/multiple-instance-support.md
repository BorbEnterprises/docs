**This is a [setting hack](all-hacks.md#setting-hacks) that can be enabled on the "Settings" page of the Mods List.**

This hack allows there to be multiple concurrently running instances of the game.

# Version History
## 1.18
Moved this hack to the new "Settings" page and made it so it's no longer hidden by default.

## 1.12
Made this hack hidden by default.

## 1.2 or earlier
Added this hack.
