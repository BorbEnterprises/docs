**This is a [developer hack](all-hacks.md#developer-hacks) that can be enabled on the "Developer" page of the Mods List.**

**This is an [advanced hack](all-hacks.md#advanced-hacks) that can be required by other hacks.**

This hack keeps a list of strings hashed by the game to allow hacks to get the string used to generate a hash for debugging purposes.
