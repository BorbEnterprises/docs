**This is an [advanced hack](all-hacks.md#advanced-hacks) that can only be required by other hacks.**

This hack provides [Lua 5.3.3](https://www.lua.org/home.html) to other hacks that want to utilise the scripting language such as [Custom Files](cf/intro.md).
