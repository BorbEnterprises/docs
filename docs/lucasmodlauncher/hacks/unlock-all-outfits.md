**This hack must be [required by a mod](all-hacks.md#mod-requirable-hacks) to be enabled.**

This hack unlocks every outfit.

# Requiring This Hack
To require this hack, add this line to your mod's Meta.ini:

```ini
RequiredHack=UnlockAllOutfits
```
