---
title: No Mission Start Cameras
description: "This hack disables mission start cameras as well as the character walking to a location at the start of a mission."
---

**This hack can be [required by a mod](all-hacks.md#mod-requirable-hacks) to be enabled.**

**This is a [setting hack](all-hacks.md#setting-hacks) that can be enabled on the "Settings" page of the Mods List.**

This hack disables mission start cameras as well as the character walking to a location at the start of a mission.

# Requiring This Hack
To require this hack, add this line to your mod's Meta.ini:

```ini
RequiredHack=NoMissionStartCameras
```
