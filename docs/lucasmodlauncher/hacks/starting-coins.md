**This is a [setting hack](all-hacks.md#setting-hacks) that can be enabled on the "Settings" page of the Mods List.**

This hack allows you to set an amount of coins you will start with when starting a new game.

# Settings
## Starting Coins
The amount of coins you will start with when starting a new game.

**Defaults to 1000.**
