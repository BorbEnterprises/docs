**This hack is [always enabled](all-hacks.md#always-enabled-hacks) when using the Mod Launcher.**

**This is an [advanced hack](all-hacks.md#advanced-hacks) that can only be required by other hacks.**

This hack allows other programs such as [Lucas' Pure3D Editor](../../lucasp3deditor/intro.md) to communicate with hacks.

# Version History
## 1.23
Made it so this hack can no longer be required by mods.

## 1.19
Fixed an issue where the Pure3D Editor would hang when selecting locators or right clicking on 3D views in some cases with the following changes:

* Made this hack still respond if the game is in a blocking modal loop or minimized while in fullscreen mode.
* Made this hack uninitialise if the game crashes.

## 1.17.1
Added this hack.
