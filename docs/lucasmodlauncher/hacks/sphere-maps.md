**This is a [setting hack](all-hacks.md#setting-hacks) that can be enabled on the "Settings" page of the Mods List.**

This hack improves reflections on cars by reimplementing the `spheremap` PDDI shader type from the console versions of the game. This reimplementation is based on the Xbox version.

By default, all `spheremap` shaders are remapped to be `environment` shaders on PC and this hack undoes that.

# Command Line Arguments
This hack is affected by certain [Command Line Arguments](../command-line-arguments.md#hack-sphere-maps) for the Mod Launcher.
