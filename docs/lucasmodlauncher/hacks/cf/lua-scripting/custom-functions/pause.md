Pauses the console and prompts the user to "Press any key to continue...".

# Syntax
```lua
Pause()
```

# Examples
```lua
Pause() -- Does as described on the tin.
```

# Notes
If the Console feature of the [Console and Logging hack](../../../console-and-logging.md) is not enabled, a console window will still be shown temporarily to prompt the user.
