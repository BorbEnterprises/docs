Returns the file name of the specified path including the extension.

# Syntax
```lua
GetFileName( <path> )
```

* **path**: The path that you want to get the file name out of.

# Examples
```lua
-- Returns "level.mfk"
GetFileName("scripts\\missions\\level01\\level.mfk")
```
