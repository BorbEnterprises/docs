**This hack is [always enabled](all-hacks.md#always-enabled-hacks) when using the Mod Launcher.**

This hack makes the game support any resolution provided by your graphics card as well as any custom resolution when it's running in a window.

# Version History
## 1.23.2
Added the `-noenumresolutions` command line argument. This stops the Mod Launcher and the hack from getting resolutions from your graphics card.

## 1.12

* Made it respect any resolution specified in `simpsons.ini` when the game is running in a Window.
* Made 1024x768, 1152x864, 1280x1024, and 1600x1200 not listed unless they're supported by your graphics card.

## 1.10
Added this hack.
