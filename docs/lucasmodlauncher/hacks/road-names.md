**This is an [advanced hack](all-hacks.md#advanced-hacks) that can only be required by other hacks.**

This hack keeps track of [Road](../../pure3d/chunk-types/chunk-3000003.md) names (which the game normally only stores as hashes) so other hacks can use them.

# Version History
## 1.23
