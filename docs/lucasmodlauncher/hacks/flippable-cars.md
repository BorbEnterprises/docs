**This is a [setting hack](all-hacks.md#setting-hacks) that can be enabled on the "Settings" page of the Mods List.**

This hack allows cars to flip over while in the air.

# Settings
## Allow Chase Cars To Flip
Set whether or not chase cars are allowed to flip over.

**Defaults to Disabled.**

# Version History
## 1.18
Moved this hack to the new "Settings" page.

## 1.12.1
Added a setting for whether or not Chase cars can flip over since the game does not reset these cars. This is disabled by default.
