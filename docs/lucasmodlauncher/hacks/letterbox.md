**This is a [setting hack](all-hacks.md#setting-hacks) that can be enabled on the "Settings" page of the Mods List.**

This hack allows you to letterbox the game to a specific aspect ratio.

# Settings
## Aspect Ratio
The aspect ratio to letterbox the game into.

## Colour
The colour of the letterbox.
