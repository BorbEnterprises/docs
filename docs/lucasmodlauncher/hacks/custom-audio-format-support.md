**This is an [advanced hack](all-hacks.md#advanced-hacks) that can only be required by other hacks.**

This hack provides functionality that allows other hacks like [FLAC Support](flac-support.md) and [OGG Vorbis Support](ogg-vorbis-support.md) to add support for custom audio formats.

# Version History
## 1.23
Made it so this hack can no longer be required by mods.

## 1.19
Added support for Lua Path Handler Redirections on RSD files to FLAC and Ogg files.

## 1.18
Added this hack.
