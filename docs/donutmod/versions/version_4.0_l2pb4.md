**This update was released on December 20th, 2019.**

# General

* Now requires Version 1.23.9 of the Mod Launcher.
* Now configures Discord Rich Presence to have more appropriate titles for the challenge missions and taxi missions.
* Added new Holiday Skin Shops that appear between December 1st and December 25th with free holiday themed skins.
	* These can be found near the start of each level.

# Level 1
Added the Milk Van and the Minivan to the Rich Side's traffic group.

# Level 2
Added idle animations to all of Bart's costumes.

# Level 2 Mission 3
Added new nerd models to add variety to the various nerd cars.

# Level 2 Mission 4

* Added an additional police blockade to the final section to prevent you from driving away.
* Added an additional stage to the final section to make it more clear where you need to go.
