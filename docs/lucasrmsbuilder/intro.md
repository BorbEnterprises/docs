Lucas' RadMusic Script Builder is a tool that allows you to convert compiled RadMusic Script files (.rms) to and from an XML format to easily edit them.

# Download
This tool is available from the [Donut Team downloads page](https://donutteam.com/downloads/RadMusicScriptBuilder/).

# Tutorials

* [Converting RadMusic Scripts to XML](tutorials/converting-rms-to-xml.md): Learn how to convert RMS files to the easily editable XML format.
* [Building RadMusic Scripts from XML](tutorials/converting-xml-to-rms.md): Learn how to build XML files back into RMS files.
