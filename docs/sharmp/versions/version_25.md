**This update was released on April 25th, 2019.**

# Launcher
## General
Now based on [Lucas' Simpsons Hit & Run Mod Launcher 1.22.4](../../lucasmodlauncher/versions/version_1.22.4.md).

## Localisation
Now supports localisation. This does not affect most of the launcher window but it does affect a few buttons, the account window and messages from Hack Support.

The Multiplayer Launcher will use your setting from the regular Mod Launcher if the selected language exists in a Languages folder next to it.

The only way to change the language setting without the regular Mod Launcher is by using the `-selectlanguage` command line argument on the Multiplayer Launcher.
