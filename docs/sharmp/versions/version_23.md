**This update was released on March 31st, 2019.**

# Launcher
## General

* Now based on [Lucas' Simpsons Hit & Run Mod Launcher 1.22.2](../../lucasmodlauncher/versions/version_1.22.2.md).
* Made it so loading a hack with `-hack` or `-hacks` will make the hack take precedence over the default hacks (like Version 1.22.1 of the Mod Launcher).
* Made the Account window show up when launching the game with no token set.
    * Only when "Play Offline" is unticked.
    * Also added "-nolaunchaccountprompt" to suppress this.
* Made everything on the window disabled and made it unable to be closed while launching the game.
