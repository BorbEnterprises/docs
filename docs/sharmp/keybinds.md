These are all of the special keybinds available when playing SHAR MP.

| Key | Function                                   |
|-----|--------------------------------------------|
| X   | Lists all players in the same level as you |
| Y   | Place a waypoint on the HUD Map            |
| >.  | Cycle Previous Character Model             |
| ?/  | Cycle Next Character Model                 |
| U   | Toggle Player Nametags                     |
| I   | Toggle Vehicle Nametags                    |
| J   | Toggle Graphical Speedometer               |
| F10 | Toggle Full-screen Map                     |
