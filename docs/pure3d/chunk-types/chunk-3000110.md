# This Chunk {{ id this }}
## Purpose
Set chunks contain multiple [Texture](chunk-19000.md) chunks inside them and can be referenced like textures inside [Shader Texture Parameter](chunk-11000.md#child-shader-texture-parameter-0x11002) chunks.

When the set is loaded, one of the textures inside is kept loaded at random and used by the game.
