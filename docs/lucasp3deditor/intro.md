Lucas' Pure3D Editor is a tool for opening Pure3D (.p3d) files found in various games created by Radical Entertainment.

The tool is primarily geared towards loading Pure3D files and chunks for The Simpsons Hit & Run with only partial support for chunk types found in other games.

# Download
## Tool
This tool is available from the [Donut Team downloads page](https://donutteam.com/downloads/Pure3DEditor4).

## Blender Add-on
This add-on is available from the [Donut Team downloads page](https://donutteam.com/downloads/BlenderXMLAddon).

As of Version 1.1.3, the addon is supported in Blender versions 2.63 through 2.82.

Prior to this version of the addon, you had to use an [unofficial port](https://donutteam.com/forum/topic/2979) in order to be able to use it Blender 2.80 or newer.
