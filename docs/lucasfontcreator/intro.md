Lucas' Pure3D Font Creator is a tool that allows you to easily generate Texture Font chunks for use in frontend files of The Simpsons Hit & Run.

# Download
This tool is available from the [Donut Team downloads page](https://donutteam.com/downloads/Pure3DFontCreator).
